# 🚀 How to Run This Project Locally

Follow these steps to set up and run the project on your local machine:

## 1️⃣ Clone the Repository
Open your terminal and run:
```bash
git clone https://github.com/Rizvi-hassan/web-atlier.git
cd web-atlier
```

## 2️⃣ Install Dependencies
Make sure you have Node.js installed, then run:
```bash
npm install
```

## 3️⃣ Start the Development Server
```bash
npm run dev
```

## ✅ You're All Set!
The application should now be running locally. Open your browser and visit:
```
http://localhost:5173
```
*(or the port shown in your terminal)*

---

## 🛠️ Prerequisites
- Node.js (v16 or higher recommended)
- npm or yarn

---

## 💡 Tips
- If you face issues, try deleting `node_modules` and running `npm install` again.
- Make sure the required port is not already in use.

Happy coding! 🎉
